How to add Favicon

First of all what is favicon? It is the little icon in browser tab next to the title of your web page. By default there is this globe grey icon. If you open your html folder in browser and have no favicon set. Look at other sites what they have set usualy it is miniature of their logo. So what to do? You have to put this code: 

<link rel="icon" href="favicon.ico" type="image/ico">
in head of your html similar like when you link CSS special font or javascript. But you also have to create that icon and save in folder with your html file. You can use this website: https://www.favicon.cc/ That will help you to create your own favicon in .ico format that you can download and use.
